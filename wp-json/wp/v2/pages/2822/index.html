FillOut Manager is a program, which enables you to fill out e.g. internet forms very fast. Drag your text directly from FOM and drop it into the edit-control you want to have filled. It’s also possible to drag&drop multiple lines of data at once.<\/p>\n

You’re probably typed in a lot of redundant information in various forms. When you use FillOut Manager, you only have to type it once.<\/p>\n

Unfortunately, there seems to be a serious problem with FillOut Manager and Windows 98 or Internet Explorer 5.0 installations. You are able to fix the above problem by hiding FOMs status bar. To do this, follow these steps:<\/em><\/p>\n

  1. Select\u00a0View<\/b>\u00a0menu<\/li>\n
  2. \n
