I had to reset my TCP\/IP stack before I got it to work which is done using<\/p>\n

C:> netsh int ip reset reset.log\n<\/code><\/pre>\n
followed by<\/p>\n
C:> netsh winsock reset catalog\n<\/code><\/pre>\n
