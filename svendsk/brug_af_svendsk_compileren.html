Brug af Svendsk compileren

Et Svendsk program kompileres til Java-kode vha. vores Svendsk-compiler svendsk. Den tager kun én parameter, nemlig selve kildefilen:

  svendsk.exe kildefil

Herefter kan den genererede java-kode kompileres videre vha. fx javac - Sun Microsystems Java Compiler.

  javac kildefil.java

Nu har man en fil i Java bytecode (en .class fil). Skal programmet eksekveres, bruges fx Suns java-fortolker java:

  java kildefil

Bemærk, at extension .class udelades i ovenstående kald.
